Cellcounter Help

Cellcounter Basics

Use File→Import Image from File... to import any image file and start marking.

To save or load images and their marks, use File→Save Image Data As..., File→Save Image Data, or File→Open Image Data...

Toolbar

Enable or Disable Mark Mode

Mouse Actions

Left-Click Mark Mode OFF: Select marker at click location
Mark Mode ON: Place a marker at click location
Right-Click Pan the image so click location is at center of window.
Mouse Scroll Wheel
or
Touchpad Scroll Vertical Gesture 		Pan the image up or down
Shift + Mouse Scroll Wheel
or
Touchpad Scroll Horizontal Gesture 		Pan the image left or right

Keystrokes

[ Zoom Out
] Zoom In
Cursor up Move view of image up
Cursor down Move view of image down
Cursor left Move view of image left
Cursor right Move view of image right
Delete
or
Backspace 		Delete selected mark(s)
